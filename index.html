hexo 常用命令

hexo new “postName” #新建文章
hexo new page “pageName” #新建页面
hexo generate #生成静态页面至public目录
hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
hexo deploy #将.deploy目录部署到GitHub
hexo help # 查看帮助
hexo version #查看Hexo的版本
hexo deploy -g #生成加部署
hexo server -g #生成加预览
hexo clean #清除缓存
命令的简写
hexo n == hexo new
hexo g == hexo generate
hexo s == hexo server
hexo d == hexo deploy
hexo c == hexo clean

hexo 如何发布新的文章到 github

一、使用hexo和github搭建完成自己的博客，那么如何发布一篇新的文章呢(hexo常用命令见：https://mrmengzb.github.io/2017/12/22/hexo-%E5%B8%B8%E7%94%A8%E5%91%BD%E4%BB%A4/)
法一
1 hexo new “postName” #新建文章（新建完成自己编辑内容）
2 hexo c #清除缓存 网页正常情况下可以忽略此条命令
3 hexo g #生成静态网页
4 hexo d #开始部署
法二
1 hexo new “postName” #新建文章（新建完成自己编辑内容）
2 hexo d -g，生成以及部署了
注意：（需要在git base 窗口中执行！！！！！）
