hexo 如何发布新的文章到 github

一、使用hexo和github搭建完成自己的博客，那么如何发布一篇新的文章呢？
法一
1 hexo new “postName” #新建文章（新建完成自己编辑内容）
2 hexo c #清除缓存 网页正常情况下可以忽略此条命令
3 hexo g #生成静态网页
4 hexo d #开始部署
法二
1 hexo new “postName” #新建文章（新建完成自己编辑内容）
2 hexo d -g，生成以及部署了
